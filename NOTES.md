## Атмосфера: Nine Inch Nails — The Downward Spiral

Сайт имитирует психологический распад. Не "страшилки", а давление.
- Тёмно-красные тени вместо ярко-красных
- Текст на странице иногда меняется на чужие слова → возвращается обратно
- Заголовок вкладки браузера периодически меняется на тревожные фразы
- Страница изредка трясётся
- Красные вспышки
- Курсор иногда "сходит с ума"
- Виньетка давит с краёв
- Скрытое послание в footer: `every day is exactly the same`
- Силуэт выглядывает из правого нижнего угла раз в ~28 сек

### Пасхалка
5 кликов по логотипу `_void` в навигации → каскад ужаса

## Как устроена мини-игра (game.js)

Холст 600×400. Управление — мышью по горизонтали.

### Зоны частоты

| Зона | % | Что происходит |
|------|---|----------------|
| Мёртвый эфир | 0–15 | Только шум, редкие сканлинии |
| Помехи | 15–30 | Размытые тёмные пятна |
| Поиск сигнала | 30–45 | Криптотексты, глитч-полосы |
| **A Warm Place** | 47–53 | Редкий момент: тишина, частицы дрейфуют вверх, "a warm place" → обрыв в красный хаос |
| Центр (сигнал) | 45–60 | Силуэты, кричащее лицо, толпа |
| Нарастание | 60–78 | Глаза, символы, руки снизу |
| Хаос | 78–100 | Всё сразу, горизонтальные разрывы, красная рамка, тряска канваса |

### Сообщения по зонам

- **Low (0–30%)**: `...`, `drift`, `hollow`, `static`
- **Mid (30–58%)**: `NOT ALONE`, `YOU OPENED IT`, `SHE STAYED`, бинарный код
- **High (58–82%)**: `THE BECOMING`, `EVERY DAY IS EXACTLY THE SAME`, `I AM NOT MY OWN`
- **Chaos (82–100%)**: `I WANT TO DESTROY SOMETHING BEAUTIFUL`, `DISSOLVING`, `THIS IS THE ONLY THING THAT'S REAL`

### Как добавить новый тип события

1. Написать функцию: `function drawMyThing(x, y, alpha) { ... }`
2. В `trySpawn()` добавить ветку с нужной зоной
3. В `renderEvents()` добавить `case 'my_thing': drawMyThing(...)`

## Хоррор-эффекты сайта (main.js — initHorror)

| Эффект | Интервал | Описание |
|--------|----------|----------|
| Красная вспышка | 7–18 сек | Кратковременный красный оверлей на страницу |
| Тряска страницы | 30–75 сек | CSS-класс `is-shaking` на `<html>` |
| Коррупция заголовка | 40–100 сек | `document.title` меняется на жуткую фразу |
| Коррупция текста | 25–60 сек | Параграф на странице меняется на чужой текст |
| Лаг курсора | 8–20 сек | Трейл курсора увеличивается и краснеет |
| Эскалация | после 3 мин | Все эффекты учащаются |

## CSS-эффекты

- `body::after` — виньетка (тёмные края)
- `body::before` — сканлиния движется сверху вниз
- `.grain` — зернистость (opacity 0.09)
- `html.is-shaking` — тряска с перекосом (skewX)
- `.section::before` — красная кровоточащая полоса слева у каждой секции
- `.void-section::after` — красный луч пересекает секцию игры
- `@keyframes body-pulse` — очень медленная пульсация фона в тёмно-красный
- `footer::before` — `every day is exactly the same` почти невидимым текстом
- Scrollbar — тёмно-красный
- Text selection — тёмно-красный фон
